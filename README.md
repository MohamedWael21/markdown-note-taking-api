# Markdown Note-taking App

An API that use to check for grammer, save, convert to html for markdown files

## Setup

make **.env** file and add the following enviroment variabels:

```javascript
TEXT_GEARS_API_KEY=
UPLOAD_FOLDER_PATH=
```

## How to use

clone repo then:

```javacript
npm install
npm run build
npm start
```
